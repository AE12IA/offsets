# FFlag offsets

This repository provides dumps of Roblox FFlag (Fast Flags) offsets for every Roblox client version.  

---

## How to Use

1. **Get Latest Offsets:**
   - Use the files in the `main` branch for the latest Roblox version.
2. **Use Offsets for an Older Version:**
   - Switch to the appropriate `version-...` branch to access offsets for that version.
3. **Query Available Versions Programmatically:**
   - Use the GitHub API:  
     ```
     https://api.github.com/repos/AE12IA/fflag-offsets/branches
     ```
   - Download a specific version’s offsets:  
     ```
     https://raw.githubusercontent.com/AE12IA/fflag-offsets/{branch}/offsets.json
     ```
